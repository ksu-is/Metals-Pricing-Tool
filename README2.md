# Frogger-Style Game (Based on Chicken Crossing Chaos)

This project is a Python-based arcade game inspired by the classic *Frogger* concept, adapted from the open source project [Chicken Crossing Chaos](https://github.com/ChillHackLab/Chicken-Crossing-Chaos). The goal is to guide your character safely across a busy road while avoiding obstacles.

Built with `pygame`, this project is part of the Sprint-based Python course at KSU.
